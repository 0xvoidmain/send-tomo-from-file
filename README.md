# Install

```
npm install
```

# Send tomo via csv file

## File format

```
Address,Tomo
0x000...0001,3
0x000...0001,3
```

## Send via command

```
node index <path_of_file>
```

## Sample

```
node index ./indexaddresses.csv
```
